# Dotfiles
<img width="1470" alt="SCR-20250117-lays" src="https://github.com/user-attachments/assets/f1a04db5-c963-4bb6-9399-0141906abba6" />
My dotfiles are currently managed with stow.

- **SKHD**: Hotkey daemon to map key combinations to opening apps, I used to use this with the yabai WM, but I switched to Aerospace.
- **Aerospace**: Window manager, I switched over from yabai because I did not want to turn off SIP for some features. I do prefer the tiling behaviour of yabai better but I rarely have more than 2 windows open on the same space at once.
- **Sketchybar**: Top bar provider. I used to use the [Uebersicht Simplebar](https://github.com/Jean-Tinland/simple-bar), but I switched to sketchy since simplebar did not have aerospace support at the time. I might switch back since simplebar has more to offer out of the box.
- **Ghostty**: Terminal, a lot to offer with zero config, and smoother scrolling behaviour with tmux and neovim.
- **Tmux**: Terminal Multiplexer, really nice to have this even though ghostty already splits windows since I ssh into machines a lot.
- **Neovim**: I use NvChad for now, might make my own config later.
